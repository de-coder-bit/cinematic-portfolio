The entire palette is controlled by CSS variables in `app/globals.css`:

  /* в”Җв”Җ Background Gradient в”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җ */
  --hero-start:   #0a0f1e;   /* в–  deep space navy  */
  --hero-mid:     #0d1b3e;   /* в–  mid navy         */
  --hero-end:     #1a2f6b;   /* в–  cobalt           */

  /* в”Җв”Җ Accent (the blue glow) в”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җ */
  --accent:       #4f8ef7;   /* в–  electric blue    */
  --accent-hover: #3a7aee;   /* в–  deeper on hover  */

  /* в”Җв”Җ Text в”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җ */

> рҹ’Ў Change `--accent` to any hex and the entire portfolio shifts colour instantly.
